Project 2

This CSS code is designed for styling a portfolio website. It includes: Body Styling: Sets the font, background color, and general layout for the entire page. Navigation Bar: The menu is styled with flexbox for horizontal alignment, and links have a hover effect with color changes and transitions. Header Styling: The header has a bright background color and a fade-in animation when the page loads. Section Styling: Sections have padding, a white background, rounded corners, and a subtle shadow, with a slide-in animation. Footer Styling: The footer has a dark background, white text, and a fade-up animation. Keyframes Animations: Used to create smooth fade-in and slide-in effects for the header, sections, and footer. Hero Image: The image is responsive, centered, and constrained to a maximum width. Text Styling: Headings, tagline, and paragraphs are styled for readability with clear font sizes and spacing.
